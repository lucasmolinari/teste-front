# TesteFront K4SW

## Dependências
 - Angular 19.1.6
 - Node 23.5
 - json-server ^1.0.0-beta.3

## Instruções
Instalar dependências:
```bash
npm i
```
Iniciar o servidor front-end:
```bash
ng serve
```
Em outro terminal, iniciar o servidor back-end:
```bash
npx json-server --watch .\assets\usuarios.json
```

Navegar para `http://localhost:4200/`
